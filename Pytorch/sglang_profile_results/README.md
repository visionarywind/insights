# Profile Results

Use `organized/` as the primary entry point.

- `organized/reports/` contains the readable comparison report and methodology.
- `organized/derived/` contains normalized summaries in `us`.
- `organized/raw/` contains original traces, console output, and raw time-ordered timelines.
- `organized/validation/` contains the minimal API profiler timeline validation data.

Older source directories are kept for traceability:

- `separate_compare_us/`
- `api_profiler_timeline_validate/`
